## 1. Requerimientos iniciales

Es necesario Quitar de disable functions:

- proc_open
- proc_close
- Correr el siguiente comando en el servidor:

```html
sudo apt-get update && sudo apt-get -y install wkhtmltopdf
```

```html
sudo apt install -y pdftk
```

## **Importante**

Antes de activar:

1. Definir en el archivo principal del plugin las constantes entre el comentario /***VARIABLES A REVISAR ANTES DE ACTIVAR EL PLUGIN***/ que sean necesarias.
2. También es necesario cambiar los correos electrónicos a los que se les enviará una copia de la solicitud de contratación. Está en la línea 655 del archivo principal del plugin.
3. Activar el plugin.
